# StockWise

Система учёта товаров на складе для курсовой работы. Десктопное приложение для управления номенклатурой, складами, документами, заказами и инвентаризацией.

## Технологии

- **.NET 10** + **Avalonia UI 11.3** (кросстплатформенный UI)
- **Entity Framework Core 10** + **Dapper** (ORM + Dapper для отчётов)
- **SQL Server** (LocalDB)
- **CommunityToolkit.Mvvm** (MVVM-архитектура)
- **BCrypt.Net** (хеширование паролей)

## Запуск

### Требования
- .NET 10 SDK
- SQL Server LocalDB

### Сборка и запуск

```bash
dotnet build src/StockWise.App/StockWise.App.csproj
dotnet run --project src/StockWise.App/StockWise.App.csproj
```

При первом запуске база данных создаётся автоматически и заполняется тестовыми данными.

### Подключение к БД

Строка подключения по умолчанию (файл `src/StockWise.App/appsettings.json`):

```
Server=(localdb)\MSSQLLocalDB;Database=StockWiseDb;Trusted_Connection=True;TrustServerCertificate=True
```

## Тестовые данные

При запуске автоматически создаются:

- **4 пользователя** (по одному на каждую роль)
- **3 контрагента**: ООО "Розница", ООО "ТехноМир", ИП "Сидоров А.В."
- **3 склада**: Основной, Склад №2, Мелкооптовый
- **11 категорий** с деревом (Электроника, Стройматериалы, Канцелярия)
- **18 товаров** в разных категориях и на разных складах
- **3 документа**: приходный, расходный и накладная на перемещение
- **3 заказа** в разных статусах (Новый, В работе, Отменён)
- **1 инвентаризация** со строками расхождений
- Остатки на складах с резервами
- Журнал транзакций

## Структура проекта

```

├── Migrations/            # Миграции EF Core

├── Repositories/          # Репозитории (Dapper)
├── Services/              # Бизнес-логика
├── ViewModels/            # MVVM ViewModels
├── Views/                 # Avalonia XAML представления
└── Program.cs             # Точка входа, DI-контейнер
```
